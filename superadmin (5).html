Procédure manuelle via Supabase Dashboard
1
Inviter l'élu par email
Supabase Dashboard → Authentication → Users → Invite user
Entre son email → il reçoit un lien pour créer son mot de passe.
→ Ouvrir Authentication
2
Attendre qu'il crée son compte
Il clique sur le lien, choisit son mot de passe. Sa ligne apparaît alors dans la table profiles.
3
Configurer son profil
Supabase Dashboard → Table Editor → profiles
Trouve sa ligne, puis renseigne :
roleadmin
commune_idid de la commune
nomPrénom Nom
pseudoprenom.nom
→ Ouvrir Table Editor
4
C'est prêt ✓
L'élu peut se connecter à la plateforme de sa commune avec son email et mot de passe.
